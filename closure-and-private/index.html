ClosureлҠ” JSмқҳ нҠ№м§•мқ„ л§җн•  л•Ң н•„мҲҳм ҒмңјлЎң лӮҳмҳӨлҠ” мҡ”мҶҢмқҙлӢӨ.

мқҙмҷҖ Typescriptмқҳ м—°кҙҖм„ұ л°Ҹ нҠ№м§•мқ„ м•Ңм•„ліҙл Өкі  н•ңлӢӨ.

нҒҙлЎңм Җ (Closure)

Goggleм—җ кІҖмғүл§Ң н•ҙлҸ„ мһ”лң©лӮҳмҷҖм„ң (Chain SmokersлҘј нҸ¬н•Ён•ҙм„ң...)

кө¬кө¬м Ҳм Ҳ м„ӨлӘ…н•ҳкё°лҸ„ лҜјл§Ңн•ҳм§Җл§Ң к°„лӢЁн•ҳкІҢ м •лҰ¬н•ҙліҙл Ө н•ңлӢӨ.

A closure is the combination of a function bundled together (enclosed) with references to its surrounding state (the lexical environment). In other words, a closure gives you access to an outer functionвҖҷs scope from an inner function. In JavaScript, closures are created every time a function is created, at function creation time. -MDN-

мүҪкІҢ л§җн•ҳл©ҙ н•ЁмҲҳм•Ҳм—җм„ң н•ЁмҲҳлҘј м •мқҳн•ҳкі  мӮ¬мҡ©н•ҳл©ҙ нҒҙлЎңм Җлқјкі  мқҙн•ҙн•ҳкі  мӮ¬мҡ©н–ҲлӢӨ.

function outter(name) {
    const welcome = `Hello ${name}`
    function inner() {
       console.log(welcome)
    }
    inner()
}

outter('akasai') // Hello akasai

мң„ мҪ”л“ңмІҳлҹј otter()н•ЁмҲҳмқҳ мҠӨмҪ”н”„лӮҙм—җм„ң inner()н•ЁмҲҳк°Җ м„ м–ёлҗҳкі  нҳём¶ңн•ҳл©° л§Ҳл¬ҙлҰ¬лҗңлӢӨ.

нҒҙлЎңм Җлқјкі  н•ҳл©ҙ к°ҖмһҘ л§Һмқҙ ліҙлҠ” мң нҳ•мқҳ мҳҲм ңмҪ”л“ңмқҙлӢӨ.


мқҖлӢүнҷ”

нҒҙлЎңм Җмқҳ нҷңмҡ©лІ•мӨ‘ к°ҖмһҘ м•Ңл Өм§„ кІғмқҖ мқҖлӢүнҷ”мқј кІғмқҙлӢӨ.

мҠӨмҪ”н”„ м•Ҳмқҳ лӮҙмҡ©мқ„ мқҖлӢүн•ҳкё° мң„н•ң мҲҳлӢЁмңјлЎң л§Һмқҙ мӮ¬мҡ©н•ңлӢӨ.

мүҪкІҢ л§җн•ҳл©ҙ private ліҖмҲҳлҘј л§Ңл“ лӢӨкі  мғқк°Ғн•ҳл©ҙ лҗ  кІғ к°ҷлӢӨ.

мқҙ кІғмқ„ кё°мЎҙ javascriptмқҳ prototypeмқ„ мқҙмҡ©н•ҙм„ң кө¬нҳ„н•ҙліёлӢӨл©ҙ м•„лһҳмҷҖ к°ҷлӢӨ.

function Count() {
    this._count = 0
}

Count.prototype.counting = function() {
    console.log(++this._count)
}

const counter = new Count();

counter.counting() // 1
counter.counting() // 2

thisнӮӨмӣҢл“ңлҘј нҶөн•ҙ prototypeмңјлЎң л¬¶кі  мқҙлҘј к°ҖмӮ°н•ҳлҠ” л°©лІ•мқҙлӢӨ.

ліҖмҲҳлӘ… м•һм—җ underscore(_)лҘј мӮ¬мҡ©н•ң _countлҠ” мқјл°ҳм Ғмқё JavaScript л„Өмқҙл°Қ м»ЁлІӨм…ҳкҙҖм җм—җм„ңлҠ” PrivateліҖмҲҳлЎң мӮ¬мҡ©н•ңлӢӨкі  н•  мҲҳ мһҲлӢӨ.

м–ҙл””к№Ңм§ҖлӮҳ м»ЁлІӨм…ҳ(м•ҪмҶҚ)мқјлҝҗ мӢӨм ңлЎңлҠ” мҷёл¶Җм—җм„ң л„Ҳл¬ҙ мүҪкІҢ м ‘к·ј/мҲҳм •мқҙ к°ҖлҠҘн•ң ліҖмҲҳлӢӨ.

function Count() {
    this._count = 0
}

Count.prototype.counting = function() {
    console.log(++this._count)
}

const counter = new Count();

counter.counting() // 1
counter.counting() // 2

counter._count = 99

counter.counting() // 100

нҒҙлЎңм ҖлҘј мқҙмҡ©н•ҳл©ҙ мўҖ лҚ” м•Ҳм •лҗң мқҖлӢүнҷ”лҘј ліҙм—¬мӨҖлӢӨ.

function Count() {
    let _count = 0
    return function() {
        console.log(++_count)
    }
}

const counting = Count()
counting() // 1
counting() // 2

мң„ мҪ”л“ңмІҳлҹј _countліҖмҲҳлҘј м ‘к·јн•  мҲҳ мһҲлҠ” лӢӨлҘё нҶөлЎңлҘј м ңкіөн•ҳм§Җ м•ҠлҠ” мқҙмғҒ мқҙлҘј м ‘к·ј/мҲҳм •н•  мҲҳ мһҲлҠ” л°©лІ•мқҖ м—ҶлӢӨ.

мҡ”м•Ҫ

к°„лӢЁн•ҳкІҢ Closureм—җ лҢҖн•ҙ м •лҰ¬н–ҲлӢӨ.

м •л§җ лҢҖлһөм Ғмқё лӮҙмҡ©мқҙкі  к№Ҡмқҙ л“Өм–ҙк°„лӢӨл©ҙ scope, л©”лӘЁлҰ¬ кҙҖлҰ¬(GC)л“ұл“ұ мӮҙнҺҙліј лӮҙмҡ©мқҙ л§Һм§Җл§Ң

м°ёмЎ°л¬ём„ңм—җ м •л§җ мһҳ м„ӨлӘ…лҗҳм–ҙмһҲлҠ” нҸ¬мҠӨнҢ…мқҙ мһҲмңјлҜҖлЎң мғқлһөн•ңлӢӨ.


Typescript

JavaмҷҖ к°ҷмқҖ мӢңмҠӨн…ңм–ём–ҙлҘј мӮ¬мҡ©н•ҙмҷ”лҚҳ л¶„л“Өм—җкІҢ мқөмҲҷн•ң private мҳҲм•Ҫм–ҙк°Җ Typescriptм—җ мЎҙмһ¬н•ңлӢӨ.

мқҙлҘј нҷңмҡ©н•ҳл©ҙ JavascriptмІҳлҹј көімқҙ нҒҙлЎңм ҖлҘј кө¬нҳ„н•ҳм§Җ м•Ҡм•„лҸ„ мқҖлӢүнҷ”лҘј н•  мҲҳ мһҲлӢӨ.


Private

private ліҖмҲҳлҘј мқҙмҡ©н•ҳм—¬ мң„ лӮҙмҡ©мқ„ кө¬нҳ„н•ҙліҙл©ҙ

class Counter {
    private count: number
    constructor() {
        this.count = 0
    }

    counting(): void {
        console.log(++this.count)
    }
}

const counter = new Counter()
counter.counting() // 1
counter.counting() // 2

л¬јлЎ ... ClassлҘј мқҙмҡ©н–Ҳм§Җл§Ң... мң„мҷҖ к°ҷмқҙ к°„лӢЁн•ҳкІҢ кө¬нҳ„н•  мҲҳ мһҲлӢӨ.

л§Ңм•Ҫ к°•м ңлЎң ліҖмҲҳм—җ м ‘к·јн•ҳл Ө н•ҳл©ҙ м•„лһҳмҷҖ к°ҷмқҖ м—җлҹ¬лҘј лұүлҠ”лӢӨ.

counter.count = 99
// TS2341: Property 'count' is private and only accessible within class 'Counter'

л„Ҳл¬ҙ нҺён•ҳм§Җ м•ҠмқҖк°Җ!? (лӘЁл‘җ TSм“°м„ёмҡ”!!!)

н•ҳм§Җл§Ң мқҳл¬ём җмқҙ мғқкёҙлӢӨ.


TSC

TypescriptлҠ” Javascriptмқҳ мғҒмң„нҷ•мһҘ(Superset)мқҙлӢӨ. кІ°көӯ мқҙ мҪ”л“ңк°Җ лҸҷмһ‘н•ҳлҠ” мӢӨм ң нҢҢмқјмқҖ .jsнҢҢмқјмқҙлӢӨ.

кіјм—° tscлЎң м»ҙнҢҢмқјлҗң jsнҢҢмқјмқҖ м–ҙл–Ө нҳ•нғңмқјк№Ң?

var Counter = /** @class */ (function () {
    function Counter() {
        this.count = 0;
    }
    Counter.prototype.counting = function () {
        console.log(++this.count);
    };
    return Counter;
}());
var counter = new Counter();
counter.counting(); // 1
counter.counting(); // 2

мӢӨм ң м»ҙнҢҢмқјлҗҳм–ҙ лӮҳмҳЁ jsнҢҢмқј(лӘ©м ҒнҢҢмқјмқҙлқјкі  н•ҙм•јн• м§Җ...)мқҖ мң„мІҳлҹј л§Өмҡ° мқөмҲҷн•ң нҳ•нғңлЎң кө¬м„ұлҗңлӢӨ.

нҒҙлһҳмҠӨлҘј мғқм„ұн•ҳлҠ” мғқм„ұмһҗ(constructor)лҠ” лҸҷмқјн•ң мқҙлҰ„мқҳ ClosureлЎң м„ м–ёлҗҳм–ҙ IIFEнҳ•нғңлЎң м •мқҳлҗҳм—ҲлӢӨ.

нҒҙлһҳмҠӨ лӮҙл¶Җ л©”мҶҢл“ңлҠ” prototypeмңјлЎң мІҙмқҙлӢқлҗҳм–ҙ мһҲм—ҲлӢӨ.

л¬јлЎ  лҚ” ліөмһЎн•ң ClassлҘј кө¬м„ұн•ҳл©ҙ jsнҢҢмқјлҸ„ лҚ” ліөмһЎн•ҙм§ҖкІ м§Җл§Ң мң„ кІ°кіјлҠ” кҪӨ мӢ м„ н•ҳкІҢ лӢӨк°Җмҷ”лӢӨ.


к·ёл ҮлӢӨл©ҙ.... мң„м—җм„ң н…ҢмҠӨнҠён•ҙліё кІғ мІҳлҹј лӮҙл¶ҖліҖмҲҳмқҳ м ‘к·ј/мҲҳм •мқҙ к°ҖлҠҘн• к№Ң?

мқҙлЎ мғҒмңјлЎ  thisм—җ л¬¶м—¬мһҲлҠ” countліҖмҲҳлҠ” м¶©л¶„нһҲ м ‘к·ј/мҲҳм •мқҙ к°ҖлҠҘн•  кІғ к°ҷмқҖлҚ°?

var counter = new Counter();
counter.counting(); // 1
counter.counting(); // 2

counter.count = 99
counter.counting(); // 100

л‘җл‘Ҙ...

prototypeмңјлЎң л¬¶м–ҙлҙӨлҚҳ мғҒлӢЁмқҳ н…ҢмҠӨнҠёмІҳлҹј лӮҙл¶ҖліҖмҲҳмқҳ м ‘к·ј/мҲҳм •мқҙ к°ҖлҠҘн•ң кІ°кіјлҘј ліҙм•ҳлӢӨ.


кІ°лЎ 

мӢӨл¬ҙм—җм„ң TypescriptлҘј мқҙмҡ©н•ҳл©ҙм„ң лҸҷлЈҢк°ңл°ңмһҗмҷҖ нҳёкё°мӢ¬м—җ мӢңлҸ„н•ҙліҙм•ҳлҚҳ лӮҙмҡ©мқ„ л°”нғ•мңјлЎң н•ҙліё н…ҢмҠӨнҠёмҳҖлӢӨ.

Typescriptм—җм„ң privateліҖмҲҳлҠ” м§„м§ң мқҖлӢүмқҙ лҗ к№Ң???

кІ°лЎ мқҖ л§һмңјл©ҙм„ң м•„лӢҲмҳҖлӢӨ.

IDEлӮҳ TsLintл“ұмқ„ мқҙмҡ©н•ҳл©ҙ м—„кІ©н•ң TSмқҳ лЈ°мқ„ м¶©л¶„нһҲ нҷңмҡ©н•  мҲҳ мһҲкі 

м• мҙҲм—җ м»ҙнҢҢмқј лӢЁкі„м—җм„ң м»ҙнҢҢмқјм—җлҹ¬лҘј ліҙм—¬мӨ„ кІғмқҙлӢӨ.

м¶©л¶„нһҲ м•Ҳм „н•ҳкІҢ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨлҠ” лң»мқҙлӢӨ.

н•ҳм§Җл§Ң мң„мҷҖ к°ҷмқҖ лӮҙмҡ©мқ„ м•Ңкі мһҲлҠ” кІғмқҙ TypescriptлҘј лҚ” м Ғм ҲнһҲ мӮ¬мҡ©н•  мҲҳ мһҲмқ„ кІғмқҙлқј мғқк°Ғн•ңлӢӨ.

лҚ•л¶„м—җ нҒҙлЎңм Җм—җ лҢҖн•ң лӮҙмҡ©лҸ„ лӢӨмӢң н•ңлІҲ м •лҰ¬н•ҙліј мҲҳ мһҲм—ҲлҚҳ кІғ к°ҷлӢӨ.
