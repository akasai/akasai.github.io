Absolute Permutation

How To Solve

단순 순열문제로 판단하고 접근했다가 테스트케이스에서 낭패를 보았다.

단순히 차이값을 판단하고 index를 구하려했는데 그런 문제도 아니였다.

개인적으로 상당히 까다로웠던 문제였다.

  1. 우선 특정 예외조건들은 따로 처리한다.

    k === 0이거나 조건이 성립할 수 없을 경우

  2. 첫 연산은 무조건 +연산만 가능하다.

    이후 발생하는 연산은 조건에 맞게 이루어진다.


Solution

function absolutePermutation (n, k) {
  const ret = []
  if (k === 0) {
    for (let i = 1; i <= n; i++) {
      ret.push(i)
    }
    return ret
  } else if ((n / k) % 2 !== 0) {
    return [-1]
  }

  let add = true
  for (let i = 1; i <= n; i++) {
    if (add) ret.push(i + k)
    else ret.push(i - k)

    if (i % k === 0) add = !add
  }
  return ret
}
